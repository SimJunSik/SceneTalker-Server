# SceneTalker-Server

### 앱 소개
---
내가 좋아하는 드라마에 대해 다른 사람들과 편하고 재미있게
감정을 공유하고 이야기를 나눌 수 있는
오직 드라마만을 다루는 하나의 공간을 마련해주는 서비스

#


### 주요 기술 스택
---
- Django-2.1.11
- Django-REST-Framework-3.10.3
- Django-Channels-2.3.0
- Channels-Redis-2.4.0
- Nginx
- Daphne
- AWS EC2
- AWS RDS MySQL
- AWS S3
